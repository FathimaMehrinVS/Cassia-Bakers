# Cassia-Bakers (Cassia Bakery ERP)

Cassia Bakers official application, built with Flutter.

---

## 🚀 Getting Started

To run this project locally, follow the steps below.

### 📋 Prerequisites

Before running the app, ensure you have the following installed on your machine:

1. **Flutter SDK**: Version compatible with Dart SDK `^3.10.7` (Flutter `3.10.x` or higher recommended).
   - [Install Flutter](https://docs.flutter.dev/get-started/install)
2. **IDE/Editor**:
   - [VS Code](https://code.visualstudio.com/) (with Flutter & Dart extensions) OR [Android Studio](https://developer.android.com/studio).
3. **Platform-specific tools**:
   - **Android**: Android Studio, Android SDK Build-Tools, and JDK.
   - **iOS / macOS**: macOS machine with Xcode and CocoaPods installed.
   - **Windows**: Visual Studio with the "Desktop development with C++" workload installed.
   - **Web**: Chrome or any modern web browser.

---

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/FathimaMehrinVS/Cassia-Bakers.git
   cd Cassia-Bakers
   ```

2. **Install Flutter packages**:
   Fetch all dependencies defined in `pubspec.yaml` by running:
   ```bash
   flutter pub get
   ```

3. **Verify environment setup**:
   Ensure your development setup is complete and recognized by Flutter:
   ```bash
   flutter doctor
   ```

4. **Run the application**:
   Start the application in debug mode:
   ```bash
   flutter run
   ```

---

## 📦 Core Dependencies

This project relies on the following package dependencies:

* **[cupertino_icons](https://pub.dev/packages/cupertino_icons)** (`^1.0.8`) - Standard iOS-style icons.
* **[mobile_scanner](https://pub.dev/packages/mobile_scanner)** (`^7.2.0`) - Universal scanner library using native camera.
* **[image_picker](https://pub.dev/packages/image_picker)** (`^1.2.2`) - Multi-platform library for selecting images/videos.
